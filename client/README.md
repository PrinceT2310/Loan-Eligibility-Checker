
# Loan Eligibility Checker

A full-stack fintech web application where users can check loan eligibility, compare banks, apply for loans, and track application status.

## Tech Stack

Frontend
- React + Vite
- Tailwind CSS
- Axios
- React Router

Backend
- Node.js
- Express.js
- MongoDB + Mongoose
- JWT Authentication
- Bcrypt

Deployment
- Frontend: Vercel
- Backend: Render
- Database: MongoDB Atlas

## Features

User Features
- Register / Login
- Loan eligibility calculator
- EMI estimation
- Bank recommendations
- Apply for loan
- Track loan applications

Admin Features
- View all loan applications
- Approve / Reject loans
- Manage banks

## Screens

- Landing page
- Dashboard
- Eligibility checker
- My applications
- Admin panel
